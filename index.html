color code

This is a digital art program made for yelp's hackathon 19 (theme: "color code"). It encodes any file into colored rectangles by converting every 3 bytes of the file into a color and optionally, the 4th byte into a width.

View on GitHub

Examples: dumb-init (w/o widths) * pip_faster.py (w/o widths) * yelp.com-logo.png (w/o widths) * pre-commit-favicon (w/o widths) * echo (w/o widths) * cat (w/o widths) * copyreg.cpython-311.pyc (w/o widths) * random (w/o widths)
